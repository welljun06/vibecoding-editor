A点击 左侧 14×14 图标方格：触发 onTogglePrimary(primary)，就地展开 / 收起当前一级。 不滚动右侧、不修改 selected 状态。
B点击 条目其余任意位置（图标右侧的标题与计数）：触发 onSelect(primary, null)， 滚动右侧到该一级的锚点，不改变展开状态。
C点击 二级 leaf：触发 onSelect(primary, secondary)， 滚动右侧到该二级锚点。
!「待接入」leaf 改成 非按钮（cursor-not-allowed），不可点击。

② 图标互换逻辑 · 行 hover 时，左侧 14×14 默认图标 → Chevron
默认态显示分类自身的图标（业务平台 / 舆情监控 等用抖音 Mark，空间用 Boxes，官方用 BadgeCheck）。当鼠标进入整行任意位置（hover），方格内图标淡出，Chevron 淡入；离开时反向。当前是否展开仅影响 chevron 朝向（右 / 下），不影响互换时机。
