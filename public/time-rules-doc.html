紧凑卡片时间展示规则

卡片元信息行中的「时间」字段，传入距今天数 (days)，按以下分段格式化为相对时间字符串。 单位仅展示最大量级（不混用 1 月 3 天前这类组合），所有非整数均向下取整 (Math.floor)。

/* 输入：自更新以来的整数天数 (days)，输出：相对时间字符串 */
days < 1 → "今天"
1 ≤ days < 7 → "N 天前" // N = days
7 ≤ days < 30 → "N 周前" // N = ⌊days / 7⌋
30 ≤ days < 365 → "N 月前" // N = ⌊days / 30⌋
days ≥ 365 → "N 年前" // N = ⌊days / 365⌋
五段分级 · 阈值与输出示例
