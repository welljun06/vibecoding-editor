紧凑卡片内容省略规则

在 200–280px 列宽区间内，卡片内任何一行都不允许换行。每行内的元素分两类： 固定不缩 (shrink-0) 与 可压缩并省略 (truncate)。 下方枚举了 4 条规则，并配以窄宽对比示例。

1标题
单行展示，超出末尾以 … 截断。不换行。
line-clamp: 1; overflow: hidden; text-overflow: ellipsis;
2标签行 (类型 / 分类 / 健康度)
行内不换行。类型 (Skill/MCP/知识库) 与 健康度 标签固定宽度，永远完整展示； 只有 分类 标签可压缩，内部文案 … 截断。
flex-wrap: nowrap; 类型/健康度 flex-shrink:0; 分类 flex-shrink:1 + truncate
3描述
单行展示，超出末尾以 … 截断。不换行。
line-clamp: 1; overflow: hidden; text-overflow: ellipsis;
4元信息行 (来源 / 时间 / 调用量)
行内不换行。时间、调用量 固定宽度，永远完整展示； 只有 来源 (用户名) 可压缩，内部文案 … 截断。
flex-wrap: nowrap; 时间/调用量 flex-shrink:0; 来源 min-width:0 + truncate
