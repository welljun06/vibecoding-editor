「紧凑卡片」自适应栅格规则

资源库右侧能力卡片网格 · 容器宽度自适应 · 列数随容器宽度变化，列宽在剩余空间内均分

/* 唯一规则 — 不设断点，由浏览器按可用宽度自动确定列数 */
display: grid;
grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
gap: 12px;

auto-fill 会塞入尽可能多的列；每列最小 200px，剩余空间通过 1fr 均分给每一列。 下面四个示例特意选在 「刚跨入下一档」的边界容器宽度，列宽落在 200–204 区间——这就是该档下能看到的最窄态。 容器继续变宽，列宽线性增加；直到下一个断点 (424 / 636 / 848 / 1060) 才会塞入新的一列。
